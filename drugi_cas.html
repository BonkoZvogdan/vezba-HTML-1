Drugi čas - Korišćenje CSS-a

 

CSS se može koristiti unutar HTML-a na tri moguća načina:

  • UMETNUTI Inline CSS - Najjednostavniji i najmanje preporučljiv način za upotrebu CSS-a je navođenje iskaza unutar STYLE atributa HTML elementa koji želimo da menjamo. Ovako kreiran kod je nečitljiv i težak za pronalaženje grešaka i izmene. Koristi se ako je potrebno na licu mesta (u samom HTML-u) izmeniti neki već postojeći CSS stil ili zadati određeni izgled bez kreiranja stila.

<a href="http://www.yahoo.com" style="color: red;">Crveni link za Yahoo</a>

  • INTERNI Embeded CSS - Bolji način od perthodnog s obzirom da odvaja prikaz od sadržine. Ipak, stilovi se i ovde kreiraju unutar same HTML stranice, s tim što su izdvojeni unutar STYLE taga unutar zaglavlja stranice. Ovaj način koristimo ako nam trebaju stilovi za koje smo sigurni da ćemo ih koristiti samo unutar te pojedine stranice.

<head>
<style>
body
{
font-family: Comic Sans MS;
background-color: #54FF9F;
color:#ffffff;
}
div
{background-color:#ffffff;
color:#00ff00;
}
</style>
</head>

  • External CSS - Izdvajanje definicija stilova u zaseban CSS fajl je najpreporučljiviji način. Ne samo da se sadržina i prikaz odvajaju u potpunosti, već se isti stilovi mogu koristiti na više različitih HTML stranica.

<head>
<link href="stil1.css" rel="stylesheet" type="text/css" />
</head>
